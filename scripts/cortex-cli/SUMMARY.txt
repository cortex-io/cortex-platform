IMPLEMENTED COMMANDS
================================================================================

1.  deploy <service>           - Deploy service to K3s cluster
2.  build <service> [tag]      - Build Docker image (+ optional push)
3.  logs <service> [-f]        - Stream logs from pods
4.  status [service]           - Show service/cluster status
5.  test <service>             - Run service tests
6.  restart <service>          - Rolling restart
7.  scale <service> <N>        - Scale replicas
8.  exec <service> [cmd]       - Execute in pod / open shell
9.  list                       - List available services
10. help                       - Show help
11. version                    - Show version

GLOBAL FLAGS:
- --namespace=<ns>             - Target namespace
- --context=<ctx>              - kubectl context
- --verbose, -v                - Debug output
